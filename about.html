Objectifs du projet

Ce projet a été réalisé dans le cadre d'un travail universitaire visant à :

  • Démontrer la maîtrise des balises sémantiques HTML5
  • Implémenter des fonctionnalités interactives avec JavaScript
  • Créer une interface utilisateur responsive et moderne
  • Appliquer les bonnes pratiques de développement web
